Overview

This document provides an overview of the multivariate relationships of all moderators with the absolute parameter deviation. As in the analysis of the univariate relationships we focus on two target methods, the complete pooling MLE method and the latent-trait partial pooling method.

In all the analyses reported here, we remove the no pooling Bayes method, as it behaved markedly different from all other methods.

Correlations and Multicollinearity (all pairs)

When looking at the multivariate relationships, we need to consider the degree of multicollinearity that could be present in the data. The following plot shows the distribution of correlations across all pairs of moderators in the data across each pair of one of the target methods with each other method. More specifically, each data point underlying the depiction in each row of the figure is the correlation for these two moderators for one pair of methods where in each case at least one of the methods in each pair is one of the two target method. The figure then depicts the median of these distribution as a solid point and the error bars depitc the minimum and the maximum.

This figure shows that for several of the pairs there is substantial correlation between .25 and .5, but for only one pair is the correlation consistently above .5, with the median just above .75. This is for the two standard errors. Therefore, in the following we combine both standard error estimates into one average.

Multiple Regression

We then consider multiple regression models.

Here, we first consider several model variants (in addition to the baseline model, mean, and the parameter model) of the nine moderators (there are ten moderators initially, and because we combine both standard error estimates we end up with nine). As further refrence models we also consider a model with both separate standard errors, mse_both (entewred as main effects) and one with only the combined standard error (mse_c). Also, for the estimate itself (y) we consider a linear and quadratic trend which is why we add two coefficients for this moderator in each model. Also, we always use the same transformation as presented in the figures in the paper (e.g., log for relative information). The following additional models are included:

  • main: Main effects only (i.e., all effects are additive)
  • main_good: Main effects of the four good moderators (SE, parameter trade-offs, relative information, and relative N).
  • inter_2: main effects and second order interactions of all moderators.
  • inter_2_godd: main effects and second order interactions of the four good moderators.
  • best2 best model with two moderators as main effects (one of which is the combined standard error)
  • best3 best model with three moderators as main effects (one of which is the combined standard error)
